# House Price Prediction using Linear Regression

## Overview
This project predicts house prices based on area using Machine Learning.

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 📊 Features
- Data visualization
- Linear regression model
- Price prediction

##  How to Run
1. Install dependencies:
   pip install pandas matplotlib scikit-learn

2. Run Jupyter Notebook:
   jupyter notebook

3. Open house_price_prediction.ipynb
